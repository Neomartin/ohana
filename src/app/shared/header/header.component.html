      Guía: Manejo de Contraseña
      Recuerda que la contraseña generada para todos los usuarios tipo "CLIENTE" es: 1234 en el caso de no recordar la contraseña de tu usuario podrás resetearla yendo a la sección Perfil.
      Guía: Añadir Libro
      Recuerda que al ingresar un nuevo libro al sistema este mismo irá buscando mientras colocas el nombre si ya existe un libro similar.
      Es importante evitar cargar libros iguales para mantener un orden en el sistema.

      TIPS: Añadir libro y buscar
      Cuando se agrega un libro al sistema es importante vincularlo de ser posible a alguna institución y asignarle carrera así como año ya que luego al buscar un archivo para agregar a la orden/tarea se podrá utilizar el código, nombre de institución, año, como palabra clave en el buscador.
